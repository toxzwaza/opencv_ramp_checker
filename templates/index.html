ランプ検知システム

OpenCVを使用したランプ色判定システムのWeb管理インターフェース

このシステムでは、カメラ画像からオレンジと緑のランプを検知し、状態を監視します。

座標設定

画像上でオレンジと緑のランプの位置を設定します。クリック・ドラッグで簡単に座標を指定できます。

座標設定を開く
設定管理

検知閾値、通知設定、カメラ設定などのシステム設定を管理します。

設定管理を開く
監視状況

リアルタイムでシステムの監視状況を確認し、監視システムの開始・停止を制御します。

監視状況を開く
分析レポート

オレンジランプの継続時間分析、ガントチャート風タイムライン、統計情報を確認できます。

  • 時系列タイムライン
  • 継続時間推移
  • 統計分析
分析レポートを開く
データ管理

検知データのダウンロード、バックアップ、システムログの確認ができます。

  • CSVデータ出力
  • データバックアップ
  • システムログ
システム情報
システム仕様
  • プラットフォーム: Raspberry Pi + Python
  • 画像処理: OpenCV
  • Web フレームワーク: Flask
  • UI フレームワーク: Bootstrap 5
主な機能
  • リアルタイムランプ色検知
  • Web ベース座標設定
  • 自動通知システム
  • 設定管理インターフェース
  • ガントチャート分析
  • 統計レポート機能
使用方法
  1. 座標設定: まず座標設定ページで、オレンジと緑のランプの位置を設定してください。
  2. 設定調整: 必要に応じて検知閾値や通知設定を調整してください。
  3. 監視開始: 監視状況ページから監視システムを開始してください。
  4. 状態確認: ナビゲーションバーの状態インジケータでシステムの動作状況を確認できます。
